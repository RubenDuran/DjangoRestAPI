Django Rest API

TranslateThis

Restful API created with Django REST Framework. Serves as the back end to TranslateThis iOS app. Uses HTTP requests to GET, PUT, POST and DELETE data.

Users

Users can be accessed via: /users

Allowing GET and POST request.

Specific User

Specific users can be accessed via: /users/USER_ID

Allowing GET, PUT and DELETE request.

Photos

Photos can be accessed via: /photos

Allowing GET and POST request.

Phrases

Phrases can be accessed via: /phrases

Allowing GET and POST request.

Specific Phrase

Specific phrases can be accessed via: /phrases/PHRASE_ID

Allowing GET, PUT and DELETE request.

Specific Photos

Specific photos can be accessed via: /photos/PHOTO_ID

Allowing GET, PUT and DELETE request.

TranslateThis. iOS app demo video.

TranslateThis is an iOS app to help translate text, voice and text in images. Developed in Swift using AVFoundation, Speech and UIKit frameworks. Application uses Google Cloud Translation API, Tesseract (OCR) and Django Backend for data retention. Users have the ability to save the output for future reference.
